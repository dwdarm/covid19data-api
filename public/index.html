COVID-19 Data API

Source: Johns Hopkins CSSE

Endpoints

URI Description
/summary Get aggregated stats for all time
/summary/latest Get latest aggregated stats
/countries Get dataset for all countries for all time
/countries/latest Get latest dataset for all countries
/countries/:country Get dataset for a specified country for all time
/countries/:country/latest Get latest dataset for a specified country
/country Get list of all available country
